# Medical Camp Management App

A Flutter application designed to streamline operations at medical camps. This app helps manage patient registration, vitals tracking, doctor assignments, and prescriptions in an organized manner.

## Features

- **Role-Based Access**: Separate login and dashboard for Volunteers.
- **Patient Management**: Easy-to-use forms for patient registration and vitals recording.
- **Workflow Management**: Streamlined process for assigning doctors and managing prescriptions.
- **Medicine Distribution**: Simple interface for medicine pickup and tracking.

## Getting Started

### Prerequisites

- Flutter SDK
- A code editor like VS Code or Android Studio
